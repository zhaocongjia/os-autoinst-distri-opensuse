DESCRIPTION

Overview

This class defines constants to use for actions:

  • YUI_PRESS - action to press a button

  • YUI_TOGGLE - action to toggle a checkbox

  • YUI_CHECK - action to check a checkbox

  • YUI_UNCHECK - action to uncheck a checkbox

  • YUI_SELECT - action to select an item

  • YUI_ENTER_TEXT - action to enter text
